# DigiTools Landing Page

A modern, responsive landing page for a digital tools marketplace. The site
highlights premium products, pricing plans, and a cart workflow with live
feedback.

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- DaisyUI
- React-Toastify
- JavaScript (ES6+)

## Key Features

- Product and cart toggle section with real-time cart counts.
- Toast notifications for add, remove, and checkout actions.
- Responsive layout with hero, stats, steps, pricing, CTA, and footer sections.
